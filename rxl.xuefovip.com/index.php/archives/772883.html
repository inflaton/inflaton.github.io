入行论第175课笔记

若触非真有，则受从何生？

何故逐尘劳，何苦伤何人？

科判
keyboard_arrow_down

摄义

善说海
keyboard_arrow_down

如果受之因——触不存在，那么其果——受又从何而生呢？何故为了追逐、获得乐受而辛苦劳累呢？究竟是什么苦受在加害何者的相续呢？所害与能害根本不存在。

字面意思
keyboard_arrow_down

参考善说海

内容分层
keyboard_arrow_down

无

逐字消文
keyboard_arrow_down

1、触：接触

2、尘劳：烦恼的异名，此处指为了追求乐受而辛苦操劳

深入分析
keyboard_arrow_down

1、如果受的因——触不存在，只是分别念假立，那虚妄的种子自然无法产生真实的果，可见快乐、痛苦、无记都是如梦如幻的没有任何实义。

2、我们没有真实的得到过任何快乐，也没有真实的感受过任何痛苦，一切都是迷乱的显现，颠倒的错觉。为了获得乐受或者摆脱痛苦而辛苦劳累，都是徒劳无益、舍本逐末的行为。

相关教证
keyboard_arrow_down

《中观宝鬘论》：虚妄之种子，所生岂能真？

《三摩地王经》：譬如有童女，夜卧梦产子，生欣死忧戚，诸法亦复然。

重点难点
keyboard_arrow_down

1、如果我们把无始以来追求快乐的时间和精力用于追求菩提，早已成就了。

2、犹如一把非常锋利的宝剑刃上面有一点点的蜂蜜，贪欲心大的人为了得到这一点点蜂蜜，用舌头去舔蜂蜜，舌头就会被割断。不明真相的世间人都是如此，说得到的利益是如此的微小，而付出的代价是如此惨重。

3、龙猛菩萨说，世间人追逐欲望的过程就像搔痒，暂时会得到舒缓、快乐，但皮肤会越来越溃烂最终丧命。智者看到这种行为会非常恐惧，所以搔痒虽然快乐，但不痒岂非更好？圣者所追求的，就是这种无漏的快乐。

消归自心
keyboard_arrow_down

不追求快乐时，当下就是快乐。不逃避痛苦时，当下就是解脱。快乐也好，痛苦也罢，全都是虚妄不实的法。

若见无受者，亦无实领受，

见此实性已，云何爱不灭？

科判
keyboard_arrow_down

摄义

善说海
keyboard_arrow_down

一旦彻见到了既无有任何受者的我，也不存在感受本身这一阶段，当时已远离了苦乐，贪爱怎么会不灭除呢？一定会灭除。

字面意思
keyboard_arrow_down

如果没有看到受苦的人，也没有真实的领受，那见到真实性之后，为什么不息灭贪爱呢？一定会息灭。

内容分层
keyboard_arrow_down

无

逐字消文
keyboard_arrow_down

1、受者：我、身体、心

2、实性：真实义

深入分析
keyboard_arrow_down

一般人的贪爱都是建立在对乐受的执着上，通过智慧分析抉择人无我或者身体不存在，从而彻底明白受者不存在，依之而产生的苦乐感受，绝对也不可能真实有。明白这样的真理后，贪爱之心怎么可能不熄灭呢！

相关教证
keyboard_arrow_down

《正法念处经》：若以正念心，如实观于色，其人于色爱，不能乱其心。

净水如意宝
keyboard_arrow_down

有情之所以无量劫来生爱不退，是因为他们内心一直都有生爱的因，也就是始终有对空性的无明，不知道受者和受无所有，才不断地爱著。如果他们有缘修学佛法，见到受者和受无自性，这“由受生爱”的妄动就会停止。

首先，因为见到无有受者我，我执所生的染污爱（凡夫）就随即消失（包括执著“我很苦”和执著“我很快乐”而生的两种染污爱都会消失）。

其次，明见没有受者的心和所受苦乐后，耽著受的非染污爱（二乘圣者）也将消失。

消归自心
keyboard_arrow_down

作为大乘修行人，我们应当对所有众生都要有爱，尽管胜义中什么执著都没有，但名言中还是要大力提倡慈悲心，这是非常重要的！

所见或所触，性皆如梦幻

科判
keyboard_arrow_down

受之对境

善说海
keyboard_arrow_down

所见与所触之间五境都是现而不实、如梦如幻的本性，为此它们的有境受也不存在实有。

字面意思
keyboard_arrow_down

参考善说海

内容分层
keyboard_arrow_down

无

逐字消文
keyboard_arrow_down

1、触：身根的对境（硬软、热冷、轻重等）

2、性：自性、本体

深入分析
keyboard_arrow_down

对方：感受的本体若不存在，见闻觉知岂不是全都不成立了？

回答：胜义中虽不成立感受者及感受的苦乐，但在二取未灭的世俗当中，如梦如幻的现相是存在的。眼睛可以看东西，耳朵可以听声音。。。虽然有种种显现，但是一经观察，全部都会息灭的。外境与有境都是如此，如同《金刚经》所说：一切有为法，如梦幻泡影。

重点难点
keyboard_arrow_down

本颂抉择的是对境无实，间接也说明了心识的本体非实有。通过分析对境不存在，能受的心也就不存在了。

净水如意宝
keyboard_arrow_down

一切法最初的生、中间的住及最后的灭都是这样虚妄无实，就如梦中的儿子最初出生、中间成长、最后死亡，这一切只是妄现。万法都如幻事、梦影、海市蜃楼。因此除了无实而现之外，真实中并无受者与所受。

比如在迷梦中明明没有色、声，没有眼、耳，还能见色、闻声。与此相同，诸法本自是梦幻体性，故能成立见、闻等。如果梦中的眼和色是实法，则无法成立以眼见色。梦中耳和声实有，也无法成立以耳闻声。正因梦中一切非实，才成立见、闻等。

消归自心
keyboard_arrow_down

虽然名言中我们都说“快乐的时候要稳重，痛苦的时候要坚强”，真正从胜义来观察，稳重和坚强都不存在，最好的方法还是要明白感受无有自性，了解到它的本体是空性。

与心俱生故，受非心能见。

后念唯能忆，非能受前心，

不能自领纳，亦非它能受。

科判
keyboard_arrow_down

执著不成立

善说海
keyboard_arrow_down

受并非是以与自身一同产生的心所能见到的，因为与心同时产生的缘故它们之间没有联系。而心在受之前或之后产生也只能忆念它、向往它而已，并不能亲自体验，因为在此等之时受尚未产生，或者已经灭亡。由于所受与能受无二，因此并不是自己领受自己。如果观察三时，则以他者来领受也不应理。

字面意思
keyboard_arrow_down

参考善说海

内容分层
keyboard_arrow_down

无

逐字消文
keyboard_arrow_down

1、俱生：同时

深入分析
keyboard_arrow_down

对方：幻化般的外境是不存在的，但感受它的心识应该存在。

回答：从以下两个角度

1）心领受苦乐时，心和受是同时还是有先后

如果同时：就成了不观待的两个法，根据因明的观点，两个同时、他体的法没有任何关系。而实际情况是，快乐和感受快乐的心是相互观待的，不可能存在一个离开心识以外单独存在的快乐或者离开快乐单独存在的乐受。

如果次第：受在前，心在后，则受已灭，无法领纳。如果受在后，也无法用现在的心感受未来的受。

结论：不观察的时候，心可以感受苦乐。一旦以理观察，心和受无论同时和不同时都无法成立。

2）受是自己领受自己，还是从他法来领受

如果是自领受：前面已经破了自证，实有的法无法安立能受、所受（如果是假立的就可以）。

如果是其他法领受：如何证明呢？又需要另一个法来领受……如此则有无穷无尽的过失。

相关教证
keyboard_arrow_down

《法集经》：说受为领纳，领纳复为谁？受者离于受，差别不可得。

重点难点
keyboard_arrow_down

全知麦彭仁波切在很多地方讲，名言当中可以有自证，自己领受自己，因为它是假立的，不是实有的自己领受自己。如果是实有的自己，又能领受自己，要么承认两个，要么承认一个。到底是一个，还是两个。如果是实有的一，不可能有能领受和所领受；如果是实有的二，绝对不是实有的一，就不是自己了。

消归自心
keyboard_arrow_down

我们何时能通达一切苦乐感受如梦如幻，何时就能对诸法不执著。就算你今天家人死了，也像是梦中死了人一样，没有什么可痛苦的，如果你在世人面前不好交代，那可以假装呜呜地哭一下；当别人办喜事非常快乐时，也像是梦中宴会一样，没有什么好快乐的，如果你实在过不了关，可以随众人笑一笑，这是证悟空性的瑜伽士随顺世人的方便方法。在他们的修行境界中，酸甜苦辣、悲欢离合只是一种模模糊糊的感觉，根本不会对它起实执。

然而，没有通达这一点的话，家人死了就会痛不欲生。不要说家人死了，就连丢了一毛钱，眼泪也会掉下来。凡夫人的执著就是这样，得到一点点时开心得不得了，失去一点点时痛苦得不得了，这就是没有认识感受的一种表现。

毕竟无受者，故受非真有，

谁言此幻受，能害无我聚？

科判
keyboard_arrow_down

执著不成立

善说海
keyboard_arrow_down

由此可见，受者终究不存在。既然受者不存在，那领受当然就不可能有了，如此一来还怎么能以乐受利益、以苦受加害这个无我的蕴聚呢？因为能利与能害均不成立。

字面意思
keyboard_arrow_down

参考善说海

内容分层
keyboard_arrow_down

无

逐字消文
keyboard_arrow_down

1、毕竟：究竟

2、聚：蕴聚

深入分析
keyboard_arrow_down

通过前面的学习，终于明白：实相中，受者不存在，与之观待的感受也不存在。既然如此，如梦如幻的感受这么可能伤害无我的五蕴集聚呢？不可能的。所利或者所害的要么是我，要么是五蕴，而我不存在，蕴也是不存在的（积聚的法都是假立）。

相关教证
keyboard_arrow_down

《正法念处经》：如实观受，不坚不实，空无所有。

龙猛菩萨：我法有与无，是事终不成，无我诸烦恼，有无亦不成。

消归自心
keyboard_arrow_down

轮回中的可怜众生，以执著“我”的缘故，对自己的感受特别在乎，从来到这个世界直至离开，一直在避苦求乐的心态中患得患失，一时不停地在奔波。现在我们通过中观的分析方法，明白了受无自性的道理，分别妄念应该会自然消失的。
